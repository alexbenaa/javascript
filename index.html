Varáveis e Tipos Primitivos
Não utilizar nomes de variaveis que: comecem com um número e não utilizar espaços.
Recomendado utilizar $ para variaveis e pode também, utilizar letras e simbulos $teste_de_variavel ou $TesteDeVariavel -> Essas são os 2 modelos mais recomendados

Dicas:
Variaveis são case sensitive
Escolha nomes coerentes
Não seja programador Alfabeto ou programador contador.

Tipos de Variaiveis:
Number -> Todos os números, tanto inteiros quanto float.
String -> Tipo de variável texto.
Boolean -> True e False.
Null -> Variável do tipo nulo, não tem valor agregado.
Undefined -> Variável indefinida.
Object -> Tipo de Objeto, pois o JS é POO.
Function -> Tipo de função, muito parecido com o Python.
Array -> Tipo de variável vetorial.

Introdução ao DOM
O que é o DOM?
R: Por ele eu consigo acessar qualquer elemento dentro da minha página. Todos os elementos são herdados conforme a estrutura do HTML, cada div cada container. window.document.write('Olá MUndo'); -> Vai escrever 'Olá Mundo ' na minha página.
window.document.write(window.document.URL) Escreve no HTML a URL.
5 Seleções que o DOM faz:

  • Marca -> getElementsByTagName()
  • ID -> getElementById()
  • Nome -> getElementsByName
  • Classe -> getElementsByClassName()
  • Seletor -> querySelector('div#id' ou 'div.class')

Aula 10

Eventos com JS.
Eventos utilizando o Mouse:

  • mouseenter -> Quando o mouse está dentro da div
  • mousemove -> Quando o Mouse está se movendo dentro da div
  • mousedown -> quando o Mouse é pressionado
  • mouseup -> quando o mouse é solto depois de pressionado
  • click -> apenas um click
  • mouseout -> Quando o mouse é retirado da div
